A tiny Java REST Client

### Maven Dependency for Consumers
Add to your `pom.xml`
```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>restiny</artifactId>
	<version>0.1.0</version>
</dependency>
```
