A micro REST Client library for Java

### Maven Dependency for Consumers
Add to your `pom.xml`
```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>microrest</artifactId>
	<version>0.1.0</version>
</dependency>
```
